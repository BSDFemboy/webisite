About Me

I am a Free Software advocate, I believe a program that doesn't respect
your privacy or freedom is a program not worth running.

My primary computer is a ThinkPad T60 of which I have replaced the boot firmware
with freeboot. I run Trisquel GNU/Linux on it because it respects my privacy.

My secondary computer is a Beelink GTR5 on which I run OpenBSD
because it is mostly free and extremely secure.
